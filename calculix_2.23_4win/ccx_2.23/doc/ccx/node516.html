searchmidneigh.f

For the smoothing procedure neighbors have to be defined for each middle node. The neighbors of a middle node are the midnodes of all elements belonging to the shell of the middle node's edge. The shell of an edge are all elements containing this edge [31]. Here, the “midnode” is used as synonym for “middle node”.
