Linear temperature function

Linear temperature function conditions can be defined between a *STEP card and an *END STEP card only.


First line:

  • *BOUNDARY,LINTEMP

Following line:

  • (global x-coefficient of normal; normal should be normalized)
  • (global y-coefficient of normal; normal should be normalized)
  • (global z-coefficient of normal; normal should be normalized)
Repeat this line if needed. 

Example:

*BOUNDARY,LINTEMP
1.,0.,0.,.1,300.,.55,650.

specifies a linear temperature function between a plane with equation (at temperature 300.) and a plane with equation (at temperature 650.).
